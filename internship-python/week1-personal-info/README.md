# Personal Information Manager

## Description
This is a beginner Python program that stores and displays personal information
using variables, user input, and formatted output.

## Features
- Takes user input
- Validates empty input
- Displays information in a clean format
- Calculates age in months

## How to Run
1. Make sure Python is installed
2. Open terminal in the project folder
3. Run:
   python personal_info.py

## What I Learned
- Python variables and data types
- Input and output
- Basic input validation
- String formatting using f-strings
